HTML Table Tag

Basic Structure

  • table: Main container for a table.
  • tr: Defines a row within the table.
  • th: Defines a header cell within a row.
  • td: Defines a standard data cell within a row.

Attributes:

  • border: Sets the width of the border around the table and cells.
  • cellpadding: Sets the space between the cell content and its borders.
  • cellspacing: Sets the space between cells.
  • width: Specifies the width of the table.
  • height: Specifies the height of the table

Accessibility:

  • summary: Provides a summary of the tableâ€™s purpose and structure for screen readers (deprecated in HTML5).
  • scope: Specifies whether a header cell applies to a row, column, or group of rows or columns.

Styling with CSS:

  • border-collapse: Controls whether table borders are separated or collapsed.
  • text-align: Aligns text within cells.
  • padding: Adds space inside each cell for better readability.

Advanced Elements:

  • < caption>: Adds a title to the table.
  • < colgroup >: Groups columns for styling.
  • < col >: Defines properties for individual columns within a colgroup.
  • < thead >: Groups header content in the table.
  • < tbody >: Groups the main body content of the table.
  • < tfoot >: Groups footer content of the table.

Summary:

  • Tables are structured using rows (< tr >) containing header (< th >) and data (< td >) cells.
  • Attributes like border, cellpadding, and cellspacing customize table appearance.
  • Accessibility is improved with attributes like scope.
  • CSS can enhance table styling, with properties such as border-collapse, text-align, and padding.
  • Advanced elements like < caption >, < colgroup >, < thead>, < tbody >, and < tfoot > help in organizing and styling tables effectively.
